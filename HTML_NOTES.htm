HTML COLOURS

  1. RGB(Red Green Blue): rgb(red, green, blue); [0 ~ 255]
  2. RGBA(RGB with Alpha channel): rgba(red, green, blue, opacity); [0 ~ 255], [0 ~ 1]
  3. HEX(Hexadecimal Colors): #rrggbb; [00 ~ FF]
  4. HSL(Hue Saturation Lightness): hsl(hue[0/360:red, 120:green, 240:blue], saturation[0~100%], brightness[0~100%]);
  5. HSLA(HSL with Alpha): hsl(hue, saturation, brightness, opacity[0~100%]);

HTML FORMS

<form name="MyForm" action="a.php" method="POST" target="_blank" enctype="text/plain">... </form>
 Method:
- POST Send the data to the header and more secure
- GET Put the data into URL and can be cache
Enctype:
- application/x-www-form-urlencoded (Default) All characters are encoded before sent (spaces are converted to "+" symbols, and special characters are converted to ASCII HEX values)
- multipart/form-data This value is necessary if the user will upload a file through the form
- text/plain Sends data without any encoding at all. Not recommended
