TECH STACK

- SASS: CSS preprocessor and will compile to CSS
    - .scss have same syntax to css and add more functionality
        - Variable, nesting, mixin(grouping attribute), function
    - .sass have similar syntax to python

- Bootstrap: CSS framework for easy designing using class

    Add this in html head:

    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

- Tailwind CSS: CSS framework for faster development, similar to bootstrap. But having more configuration space

    https://tailwindcss.com/

- Purge CSS: remove unused CSS using PostCSS plugin

    https://purgecss.com/getting-started.html

- MongoDB: storage on document; SQL: storage on relations

- Typescript: a language derived from javascript to deal with variable type consistency. Compiles to javascript.
