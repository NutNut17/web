CSS INTRODUCTION

Cascading Style Sheets (CSS) use .css as file extension. CSS can be added to HTML documents in 3 ways:
Inline - by using the style attribute inside HTML elements
Internal - by using a "style" element in the <head> section
External - by using a "link" element to link to an external CSS file <link rel="stylesheet" href="/Personal HTML Notes/ExternalFiles/HTMLstyles.css">

CSS LAYOUT PROPERTY

Layout Description Value
display whether the element will be show on the webpage none, block, inline, inline-block, flex, list-item
visibility whether the element will be visible on the webpage visible, hidden, collapse
position position of specific element static(default), relative(add top:a px; left:b px;), absolute
z-index the higher the value, the more privilege it gets integer
overflow way to handle if the element overflows size, visible, hidden, scroll
box-shadow shadow of the element offset-x offset-y blur-radius color
float float the elemtent to a direction none, left, right, initial, inherit
clear remove float property none, left, right
justify-content aligns the flexible container's items when the items do not use all available space on the main-axis center, flex-start, flex-end, space-between, space-around, space-evenly
align-items align the items vertically center, stretch, flex-start, flex-end
