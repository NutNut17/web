JQUERY INTRODUCTION

JQuery is a Javascript Library must be declared in script tag.

Feature Syntax Description
Selector
Basic Select $("selector") Selector is the same as HTML selector including pseudo class
Input Select $("input[name='id']") Select input element with name as id
Get and Set Function
text .text() / .text("value") Get or set the text value
HTML .html() / .html("value") Get or set the html value
value .val() / .val("value") Get or set the val value
CSS .css("property", "value") Access single CSS properties
CSS .css({"prop1":"value","prop2":"value"}) Access multiple CSS properties
prop .prop("property", value) Access property
append .append("<li> List </li>") Append list to the selected element
remove .remove() Remove the element
empty .empty() Empty the element
show / hide .show() / .hide() Show or hide the element
toggle .toggle() Toggle the show/hide element
removeAttr .removeAttr('attributename') Remove the attribute
mouseover / mouseout .mouseover(func...) / .mouseout(func...) Run the function when mouseover or mouseout
hover .hover(fun1(mouseover), fun2(mouseout)) Run the fun1 or fun2 according to mouse location
change .change(func...) Run the function when an event occurs
click .click(func...) Run function when clicked
