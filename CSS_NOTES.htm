CSS INTRODUCTION

Cascading Style Sheets (CSS) use .css as file extension. CSS can be added to HTML documents in 3 ways:
Inline - by using the style attribute inside HTML elements
Internal - by using a "style" element in the <head> section
External - by using a "link" element to link to an external CSS file <link rel="stylesheet" href="/Personal HTML Notes/ExternalFiles/HTMLstyles.css">
